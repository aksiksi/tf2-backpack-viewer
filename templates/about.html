About


This website started out as a weekend Python project. It was created by Assil Ksiksi (@Cyph0n) to aid the fellow in improving his Python and Flask programming. The template (or theme) used is Twitter Bootstrap, an extremely flexible and well-documented UI. The project in its entirety can be found on the creator's GitHub page.

How?

The app itself is, as mentioned, written in the Flask web framework for Python. I'd like to roughly outline what the app goes through each time a request is made, to give an idea of the inner workings of the app to anybody who is interested.

  1. The app gets (no pun intended) the search query, the dropdown menu option, and the state of the checkbox.
  2. Depending on the dropdown option, a function converts (or doesn't) the input to SteamID64 format, because this is what the Steam API requires.
  3. If the SteamID is new i.e. the app hasn't served it before, a request is made to the Steam API, which returns the user's backpack contents.
  4. If it isn't new, the app instead gets the item response from the cache, under the condition that the data is no older than 3 days.
  5. Next, the item response is passed to another function, where it is parsed and formatted, so it can be nicely outputted in HTML.
  6. Finally, all the processed data is passed to the template, where Jinja2, a templating engine, allows everything to be nicely outputted to the page.
